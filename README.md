## 🌟 简介

LifeHabit 是一款基于科学行为设计理论的习惯养成应用，帮助您轻松培养良好的生活习惯。我们的核心理念是"从小开始，持续进步"，通过微习惯系统让习惯养成变得简单而有效。

---

## ✨ 核心特色

### 🎯 **两阶段微习惯系统**

- **第一阶段**：设置极其简单、100%可完成的目标
- **第二阶段**：在舒适区内逐步增加难度

### 📊 **数据驱动的习惯构建**

- 从时间、持续时长、频率三个维度追踪习惯
- 智能数据分析帮助您找到最优目标设置
- 基于真实表现数据逐步调整习惯难度

### 🎮 **游戏化激励系统**

- 为自己设置奖励，用打卡获得能量兑换
- 超过70枚原创手绘打卡图标
- 精美分享图让每一个进步都值得庆祝

### 🔔 **智能提醒系统**

- 多种提醒方式，不给偷懒找借口
- 桌面小组件：打卡、奖励、统计一应俱全
- 每日锦囊：基于科学指导的温暖鼓励
